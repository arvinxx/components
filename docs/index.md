# 说明

该仓库是我自行积累的设计组件库

## 为什么会有这个?

- 做过的设计挺多, 但是缺少总结
- 业务组件如果可以沉淀,那么未来可以给自己复用
- 终身计划的一环

首要目的不是给外部使用,而是作为我个人设计资源的整理
